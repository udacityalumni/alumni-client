## Avatar Component
A component that that displays a user avatar.

### Example

```js
<Avatar src={user.avatar} />
```

### Props

| Prop          | Type     | Default     | Possible Values
| ------------- | -------- | ----------- | ---------------------------------------------
| **src**    | String   | defaultAvatarUrl            | Any string value


### Other Information
Presentational component that can be reused if needed.
